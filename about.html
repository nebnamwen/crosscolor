crosscolor

about this game

crosscolor is a remake, created by Benjamin Newman and Claude Code, of Blendoku by Lonely Few. Blendoku hasn't been available on the Android or iOS app stores for a few years now, so creating a remake is one way I can share this game concept (which I love) with my friends. If you like this game, and if the original ever becomes commercially available again, I encourage you to support the original creators by buying it.

how to play

Select a puzzle shape from the menu. Puzzle shapes are divided into tabs by difficulty — more shapes and difficulty tiers may be added later.

Click a tile to select it and click again to place it or swap it with another tile. Place tiles from the palette into the puzzle grid so that all straight lines form smooth color gradients. Win by placing all tiles correctly, or win perfectly by placing all tiles without placing any in an incorrect grid cell along the way.

how this was made

crosscolor was built in collaboration with Claude Code as an exercise in Spec-Driven Development — this was a highly interactive process in which human and AI both contributed to a specification document describing all key features and the plan for implementing them. These artifacts are preserved in the project's git repository. AI wrote the vast majority of the code, while I reviewed and tested after each implementation step. No graphical assets were created using AI — in fact the game has no graphical assets, it's all done with HTML and CSS.
